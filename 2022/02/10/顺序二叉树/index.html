static class Tree {

2.2.4 树深度

返回一个树的深度，与遍历极为类似，稍微改点东西。

/**
* @param father 从此节点开始计算深度。
* @return  father节点开始的深度
*/
public static int depthTree(Tree father) {
    if (father.right == null && father.left == null) {
        return 0;
    } else {
        int left = 1, right = 1;
        if (father.left != null) {
            left += depthTree(father.left);
            //本层加上子树的最大层
        }
        if (father.right != null) {
            right += depthTree(father.right);
            //本层加子树的最大层
        }
        return Math.max(right, left);
        //返回左右树中的极大值，如果相等，则返回右值（右值也是一，表示当前层）
    }
}

节点数统计方法类似，此处偷个懒。

2.2.5 遍历树

三种方式，先序、中序、后序，这里通过一个枚举类型进行选择。

LDR: 从小到大顺序；

另外两种比较好观察数据树的结构。

/**
     * @param father    起始节点
     * @param treeNodes 一个object类型的列表，用于接收遍历数据
     * @param trOr      遍历方法，DLR，LDR，LRD 
     */
public static void traverseTree(Tree father, List<Object> treeNodes, traverseOrder trOr) {
    switch (trOr) {
        case DLR: {
            //前序遍历
            if (father.data != -1) {
                treeNodes.add(father.data);
            }
            if (father.left != null) {
                traverseTree(father.left, treeNodes, trOr);
            }
            if (father.right != null) {
                traverseTree(father.right, treeNodes, trOr);
            }
            break;
        }
        case LDR: {
            //中序遍历
            if (father.left != null) {
                traverseTree(father.left, treeNodes, trOr);
            }
            if (father.data != -1) {
                treeNodes.add(father.data);
            }
            if (father.right != null) {
                traverseTree(father.right, treeNodes, trOr);
            }
            break;
        }
        case LRD: {
            //后序遍历
            if (father.left != null) {
                traverseTree(father.left, treeNodes, trOr);
            }
            if (father.right != null) {
                traverseTree(father.right, treeNodes, trOr);
            }
            if (father.data != -1) {
                treeNodes.add(father.data);
            }
            break;
        }
        default: {
            //实际运行不到这里。
            System.out.println("遍历方向有误，将默认采用中序遍历");
            traverseTree(father, treeNodes, traverseOrder.LDR);
            break;
        }
    }
}

2.2.6 查找节点

通过遍历，再根据二叉树的特性，值小向左，值大向右。

找到时返回节点。

为了方便对节点的父节点进行操作，为节点设置了父节点属性。

使用比较多，原理也比较简单。

/**
 * @param father    根节点
 * @param nodeValue 查找的值
 * @return 返回值所在节点。如果不存在则返回最后null。
 */
public static Tree findNode(Tree father, int nodeValue) {
    if (father.data == nodeValue) {
        return father;
    } else {
        while (father.left != null || father.right != null) {
            if (nodeValue < father.data && father.left != null) {
                father = father.left;
            } else if (nodeValue > father.data && father.right != null) {
                father = father.right;
            } else {
                break;
            }
        }
    }
    if (father.data != nodeValue) {
        //到底了没找到。
        return null;
    } else {
        return father;
    }
}

2.2.7 插入

2.2.7.1 插入值

根据值通过递归找到合适位置。然后新建节点，并连接。

此处的连接是双向的，因为存在父节点。

/**
     * @param father 根节点
     * @param nodeValue 插入到合适位置的数据
     */
public static void insertNode(Tree father, int nodeValue) {
    if (father.data == -1) {
        father.data = nodeValue;
    } else {
        //非空树
        if (father.data > nodeValue) {
            //左枝
            if (father.left != null) {
                //存在节点继续
                insertNode(father.left, nodeValue);
            } else {
                father.left = new Tree(nodeValue);
            }
        } else {
            //右枝
            if (father.data < nodeValue) {
                if (father.right != null) {
                    //同理
                    insertNode(father.right, nodeValue);
                } else {
                    father.right = new Tree(nodeValue);
                }
            }
        }
    }
}
2.2.7.2 插入节点

public static void insertNode(Tree father, Tree ele) {}

father 附加的目的树。

ele 为一个树的根节点，即将一个树附加到另一个树的合适位置上。

主要操作为：

  1. 判断是否为空树。

    为空：

    1. 将节点的数据赋值给目的节点。
    2. 修改附加节点的根节点的父节点（如果存在的话）。

  2. 通过递归根据子节点的值找到合适位置。

  3. 插入值。

在删除一个节点的方法中也有用到此方法。

/**
     * @param father 根节点
     * @param ele 插入节点
     */
public static void insertNode(Tree father, Tree ele) {
    if (father.parNode == null && father.data == -1) {
        father.data = ele.data;
        father.left = ele.left;
        father.right = ele.right;
        ele.parNode = father;
        if (ele.left != null) {
            ele.left.parNode = father;
        }
        if (ele.right != null) {
            ele.right.parNode = father;
        }  
    } else {
        //非空树
        if (father.data >= ele.data) {
            //左枝
            if (father.left != null) {
                //存在节点继续

                insertNode(father.left, ele);
            } else {
                father.left = ele;
                ele.parNode = father;
            }
        } else {
            //右枝
            if (father.data < ele.data) {
                if (father.right != null) {
                    //同理
                    insertNode(father.right, ele);
                } else {
                    father.right = ele;
                    ele.parNode = father;
                }
            }
        }
    }
}

2.2.8 删除

2.2.8.1 删除节点

比较复杂的操作，难点在于考虑删除节点的上下情况。

如下为我的分类讨论（熟悉的名词）：

先定义几个量：

deleteNode :要删除的节点。

left ：删除节点的左节点属性。

right： 删除节点的右节点属性。

parNode： 删除节点类的父节点属性。

parentOfNode: 删除节点的父节点。

① 叶节点。

② 非叶节点

  1. 根节点

    1. 左子树为空

    2. 右子树为空或两树都不为空。

      ​ 之所以把 仅右子树为空 与 两树都不为空 放一起，是因为有个操作可以简化过程。

      ​ 原来过程分三种：

      • deleteNode.left==null ，deleteNode.right!=null。

        操作：用deleteNode.right 的值覆盖deleteNode

      • deleteNode.left!=null，deleteNode.right==null。

        操作：用deleteNode.left 的值覆盖deleteNode

      • deleteNode.left!=null，deleteNode.right!=null。

        操作：用deleteNode.left 的值覆盖deleteNode，然后将deleteNode.right 子树附加到deleteNode.left上，附加操作可使用insertNode()

        deleteNode.left==null且 deleteNode.right==null 为叶节点，最开始已排除。

  2. 非根节点

    1. …同上的分类，但是多一个操作，将父节点向下的连接进行修改（需要判断是左枝到deleteNode，还是右枝）

最后是一个输出此次删除操作消失的节点值，用于调试以及删除树操作的显示。

/**
* @param father 操作的树
* @param value  根据值删除第一次出现的节点
*/
public static void deleteNode(Tree father, int value) {
    //未删除时根元素的全部元素
    List<Object> nodeData = new ArrayList<>();
    traverseTree(father, nodeData, traverseOrder.LDR);

    Tree term = findNode(father, value);
    if (term == null) {
        System.out.println(String.format("未找到:%3d ", value));
        return;
    }
    if (term.left == null && term.right == null) {
        // 叶节点。 删除节点的左右节点都为空。
        term.left = null;
        term.right = null;
        term.data = -1;
    } else {
        //非叶节点
        if (term.parNode == null) {
            //删除根节点
            if (term.left == null) {
                //左子树为空
                term.data = term.right.data;
                //根节点的 左树的 左节点，左树的 右节点
                Tree rLOfTerm = term.right.left, rROfTerm = term.right.right;
                //换子节点
                term.right = rROfTerm;
                term.left = rLOfTerm;
                //换父节点
                if (rLOfTerm != null) {
                    rLOfTerm.parNode = term;
                }
                if (rROfTerm != null) {
                    rROfTerm.parNode = term;
                }           
            } else {
                //右子树为空,或两子树都在。
                //根节点的 左树的 左节点，左树的 右节点
                Tree lLOfTerm = term.left.left, lROfTerm = term.left.right;
                //换子节点
                term.data = term.left.data;
                term.left = lLOfTerm;
                Tree deleteNodeTermRight = term.right;
                term.right = lROfTerm;
                if (deleteNodeTermRight != null) {
                    insertNode(term, deleteNodeTermRight);
                }
                if (lLOfTerm != null) {
                    //左子树的左枝不为空。
                    lLOfTerm.parNode = term;
                }
                if (lROfTerm != null) {
                    //左子树的右枝不为空。
                    lROfTerm.parNode = term;
                }
            }
        } else {
            //删除非根节点。
            //定位节点 父节点 左 右子树
            Tree parOfTerm = term.parNode, lNode = term.left, rNode = term.right;
            if (term.left == null) {
                //删除节点左子树为空。
                //换父节点
                rNode.parNode = parOfTerm;
                if (parOfTerm.left == term) {
                    parOfTerm.left = rNode;
                } else {
                    parOfTerm.right = rNode;
                }
            } else {
                //删除节点左子树为空，或两子树都在
                //换父节点
                lNode.parNode = term.parNode;
                //换子节点
                if (parOfTerm.left == term) {
                    //当前节点为父节点左枝，赋值
                    parOfTerm.left = lNode;
                } else {
                    //...
                    parOfTerm.right = lNode;
                }
                if (term.right != null) {
                    //将 右枝 根节点 插入到左枝中--子节点连带过去。
                    insertNode(term.left, term.right);
                }
            }
        }
    }

    System.out.println(String.format("删除%3d,", value));
    //删除后的所有元素
    List<Object> newNodeData = new ArrayList<>();
    traverseTree(father, newNodeData, traverseOrder.DLR);

    System.out.print("\t删除元素: ");
    //原树的补集
    Object[] arr3 = nodeData.stream().filter(elem -> !newNodeData.contains(elem)).toArray();
    for (Object o : arr3) {
        //遍历输出
        System.out.print(o + " ");
    }
    System.out.println();
}
2.2.8.2 删除子树

较上一个更简单，直接使定位到的节点的各值为空节点即可。

/**
* @param father 操作的树
* @param value  根据值删除第一次出现的节点的子树
*/
public static void deleteTree(Tree father, int value) {   
    Tree term = findNode(father, value);
    if(term!=null){
        //未删除时根元素的全部元素
	    List<Object> nodeData = new ArrayList<>();
    	traverseTree(father, nodeData, traverseOrder.LDR);
        //存在此子树
        term.left = null;
        term.right = null;
        term.data = -1;
        System.out.print(String.format("删除%3d 为根节点的树,", value));
        //删除后的所有元素
        List<Object> newNodeData = new ArrayList<>();
        traverseTree(father, newNodeData, traverseOrder.DLR);
        System.out.print("删除的子树节点: ");
        //原树的补集
        Object[] arr3 = nodeData.stream().filter(elem -> !newNodeData.contains(elem)).toArray();
        for (Object o : arr3) {
            //遍历输出
            System.out.print(o + " ");
        }
        System.out.println();

    }else{
        System.out.println(String.format("未找到子树:%3d", value));
    }
}
